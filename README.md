# Spotify Track Feature Analysis (Power BI)

This project analyzes Spotify track audio features and track popularity using SQL, Excel, and Power BI.

## Tools Used
- SQL (data cleaning)
- Excel (descriptive statistics)
- Power BI (dashboard + visuals)

## Repo Structure
- `powerbi/` → Power BI report (.pbix)
- `data/cleaned/original/` → cleaned dataset or dataset (.csv)
- `sql/` → SQL scripts used to clean/filter the dataset
- `python/` → Python scripts
- `docs/` → documentation + screenshots of code
- `screenshots/` → dashboard images

## Key Findings
- Mean popularity: 33.23 (Spotify popularity score 0–100)
- Median popularity: 34
- Standard deviation: 22.32
- Genres such as pop-film and k-pop show higher average popularity.
